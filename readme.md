# Task Manager CLI

Простой CLI task manager на Python с JSON-хранилищем.  

## Возможности
- Добавление задач
- Просмотр всех / выполненных / невыполненных задач
- Удаление задач
- Обновление задач (включая статус выполнения)
- Логирование действий через декораторы
- Фильтры и сортировка задач

## Как запустить
1. Убедитесь, что Python >= 3.10 установлен
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/username/task-manager-cli.git
